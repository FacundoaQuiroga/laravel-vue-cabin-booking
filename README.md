# Cabin Reservation Platform

A modern full-stack cabin reservation management system built with Laravel 12 and Vue 3.

## Features

- User authentication
- Role-based authorization
- Cabin management
- Reservation management
- Availability calendar
- RESTful API
- Responsive UI
- Dockerized development environment

## Tech Stack

- Laravel 13
- Vue 3 Composition API
- Pinia
- Axios
- MySQL
- Docker
- Nginx
